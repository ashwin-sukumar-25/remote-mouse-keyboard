# Remote Mouse & Keyboard Controller

A cross-platform system that lets you control your PC mouse and keyboard using a mobile phone.

## Tech Stack
- Flutter – Mobile app
- Java – Desktop server (Windows, Linux, macOS)

## Features
- Remote mouse movement
- Keyboard input from phone
- Cross-platform Java backend
- Simple and fast connection

## Project Structure
- flutter app/ → Mobile client
- java server/ → Desktop server

## Setup
1. Run the Java server on your PC
2. Launch the Flutter app on your phone
3. Connect both on the same network
4. Start controlling your PC
